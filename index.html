🏗️ CSS განლაგების ისტორია

CSS განლაგების ევოლუციის გასაგებად მნიშვნელოვანია ვიცოდეთ როგორ განვითარდა ეს ტექნოლოგიები დროის მიმდინარეობაში:

🕰️ დროის ხაზი:

1990s-2000s: Table-based layouts და Float positioning
2009: CSS3 Flexbox-ის შემოღება (მაგრამ არც ისე პოპულარული)
2012: CSS Grid-ის პირველი სპეციფიკაცია
2017: CSS Grid-ის ფართო ბრაუზერული მხარდაჭერა
დღეს: Modern CSS Layout System - Grid + Flexbox + Position

💡 რატომ არის ეს მნიშვნელოვანი?

თანამედროვე CSS layout ტექნიკები გვიჩვენებს:

  • მოქნილობა: შეგვიძლია შევქმნათ რთული განლაგებები მარტივი კოდით
  • Responsive Design: ადვილად ვადაპტირდებით სხვადასხვა ეკრანის ზომებზე
  • მანტენერებადობა: კოდი უფრო სუფთა და გასაგებია
  • პერფორმანსი: ბრაუზერები ოპტიმიზებულია ამ ახალ ტექნიკებისთვის

🎨 CSS Box Model - ფუნდამენტური ცნებები

ყველა CSS layout ტექნიკის გასაგებად, პირველ რიგში უნდა ვიცოდეთ როგორ მუშაობს CSS Box Model:

Content
Margin
Border
Padding

Box Model-ის კომპონენტები:

.box-model-example { width: 200px; /* Content area width */ height: 100px; /* Content area height */ padding: 20px; /* შიდა ღია ადგილი */ border: 5px solid red; /* საზღვრები */ margin: 10px; /* გარე ღია ადგილი */ } /* Total width = width + padding-left + padding-right + border-left + border-right + margin-left + margin-right */ /* Total width = 200px + 20px + 20px + 5px + 5px + 10px + 10px = 270px */

Box-sizing Property:

/* Default behavior (content-box) */ .content-box { box-sizing: content-box; /* width/height მხოლოდ content-ს ითვალისწინებს */ } /* Modern approach (border-box) */ .border-box { box-sizing: border-box; /* width/height შეიცავს content + padding + border */ } /* Universal border-box (რეკომენდებული) */ *, *::before, *::after { box-sizing: border-box; }

📏 CSS Units - ზომების ერთეულები

სწორი units-ების არჩევა კრიტიკულია Responsive Design-ისთვის:

🔢 Absolute Units (აბსოლუტური):

px (pixels)
ყველაზე ხშირად გამოყენებული, ზუსტი კონტროლისთვის
pt (points)
ძირითადად print media-სთვის
cm, mm, in
ფიზიკური ზომები, იშვიათად გამოიყენება

📐 Relative Units (ფარდობითი):

em
ფარდობითი მშობელი ელემენტის font-size-თან
/* თუ parent-ის font-size: 16px */ .child { font-size: 1.2em; } /* = 19.2px */
rem
ფარდობითი root ელემენტის font-size-თან
/* თუ html font-size: 16px */ .element { font-size: 1.5rem; } /* = 24px */
vw, vh
Viewport width/height-ის პროცენტი
.full-screen { width: 100vw; /* full viewport width */ height: 100vh; /* full viewport height */ }
% (percentage)
ფარდობითი მშობელი ელემენტის ზომასთან
.half-width { width: 50%; /* parent-ის width-ის ნახევარი */ }

🎯 რომელი Units გამოვიყენოთ რაისთვის:

  • px: borders, shadows, small fixed elements
  • rem: font-sizes, margins, paddings (main choice)
  • em: components რომლებიც იცვლება მშობელი font-size-ის მიხედვით
  • vw/vh: hero sections, full-screen elements
  • %: responsive widths, container-based sizing

🎭 CSS Selectors - მნიშვნელობა და Specificity

CSS-ში სელექტორების სწორი გამოყენება განსაკუთრებით მნიშვნელოვანია რთული პროექტებში:

🎯 Specificity Hierarchy (მნიშვნელობის იერარქია):

1000

Inline styles

style="color: red"

100

IDs

#header

10

Classes, Attributes, Pseudo-classes

.menu, :hover

1

Elements, Pseudo-elements

div, ::before

📊 Specificity გამოთვლის მაგალითები:

/* Specificity: 0,0,0,1 (1) */ h1 { color: blue; } /* Specificity: 0,0,1,1 (11) */ h1.title { color: red; } /* Specificity: 0,1,1,1 (111) */ #header h1.title { color: green; } /* Specificity: 1,0,0,0 (1000) */ <h1 style="color: purple;">...</h1> /* !important ყველაზე ძლიერია (მაგრამ ავოიდ კეთება რეკომენდებული) */ h1 { color: orange !important; }

🎨 Best Practices:

  • არ გამოიყენოთ !important - გამონაკლისის გარდა
  • ID-ების ნაცვლად Classes გამოიყენეთ - უფრო მოქნილია
  • BEM methodology - Block Element Modifier
  • CSS Architecture - OOCSS, SMACSS, ITCSS
/* BEM Methodology Example */ /* Block */ .card { } /* Element */ .card__header { } .card__body { } .card__footer { } /* Modifier */ .card--large { } .card--featured { } .card__header--centered { }

🌍 Modern CSS Features Overview

ამ სესიაში ყველა თემა ერთმანეთს უკავშირდება და ქმნის ძლიერ განლაგების სისტემას:

🏗️ Layout Systems

CSS Grid + Flexbox ერთად ქმნიან ძლიერ layout სისტემას:

  • Grid - 2D layouts (rows + columns)
  • Flexbox - 1D layouts (ერთი მიმართულება)
  • Position - Precise control

🎬 Animation & Interaction

Transform + Transition + Animation ქმნიან interactive experiences:

  • Transform - Shape & position changes
  • Transition - Smooth property changes
  • Animation - Complex keyframe sequences

🎨 Theming & Variables

CSS Variables საშუალებას გვაძლევს ვქმნათ maintainable themes:

  • Global design tokens
  • Dynamic theme switching
  • Component-based styling

📱 Responsive Design

Mobile-first approach + Grid + Flexible units:

  • Media queries strategically
  • Flexible grid systems
  • Scalable typography

🔗 როგორ ვაკავშიროთ ეს ყველაფერი:

/* Modern CSS Architecture Example */ :root { /* Design Tokens */ --primary-color: hsl(210, 50%, 50%); --spacing-unit: 1rem; --border-radius: 0.5rem; --animation-duration: 0.3s; } .card { /* Layout with Grid */ display: grid; grid-template-rows: auto 1fr auto; /* Styling with Variables */ background: var(--card-bg, white); border-radius: var(--border-radius); padding: var(--spacing-unit); /* Animation */ transition: transform var(--animation-duration) ease; } .card:hover { /* Transform on interaction */ transform: translateY(-2px) scale(1.02); } @media (max-width: 768px) { /* Responsive adjustments */ .card { grid-template-rows: auto; } }

📐 CSS Grid Layout Module

CSS Grid არის ორგანზომილებიანი განლაგების სისტემა, რომელიც საშუალებას გაძლევთ შექმნათ რთული განლაგებები რიგებითა და სვეტებით.

ძირითადი ცნებები:

  • Grid Container - მშობელი ელემენტი display: grid-ით
  • Grid Items - კონტეინერის პირდაპირი შვილები
  • Grid Lines - განყოფილებები რომლებიც ქმნიან სტრუქტურას
  • Grid Tracks - ადგილი ორ მეზობელ ხაზს შორის

Grid Container Properties:

.grid-container { display: grid; grid-template-columns: repeat(3, 1fr); /* 3 თანაბარი სვეტი */ grid-template-rows: 100px auto 50px; /* 3 რიგი კონკრეტული სიმაღლით */ gap: 20px; /* ადგილი ელემენტებს შორის */ grid-template-areas: "header header header" "sidebar main main" "footer footer footer"; }

Grid Item Properties:

.grid-item { grid-column: 1 / 3; /* გაშლა 1-დან 3-მდე სვეტამდე */ grid-row: 2 / 4; /* გაშლა 2-დან 4-მდე რიგამდე */ grid-area: header; /* სახელდებული ზონის გამოყენება */ }

📍 CSS პოზიციონირება

CSS პოზიციონირება აკონტროლებს თუ როგორ იქნება ელემენტები განლაგებული მათ მშობელ კონტეინერში.

Static (ნაგულისხმევი):

.static { position: static; } /* ელემენტები ბუნებრივად იშლებიან დოკუმენტში */

Relative:

.relative { position: relative; top: 10px; left: 20px; } /* პოზიცია ფარდობითია თავისი ბუნებრივი პოზიციისადმი */

Absolute:

.absolute { position: absolute; top: 0; right: 0; } /* პოზიცია ფარდობითია უახლოესი positioned წინაპრისადმი */

Fixed:

.fixed { position: fixed; bottom: 20px; right: 20px; } /* პოზიცია ფარდობითია viewport-ისადმი */

Sticky:

.sticky { position: sticky; top: 0; } /* იცვლება relative-სა და fixed-ს შორის scroll-ის მიხედვით */

🔄 CSS Transform Property

Transform საშუალებას გაძლევთ შეცვალოთ ელემენტების კოორდინატების სისტემა ბუნებრივი დოკუმენტის ნაკადის დარღვევის გარეშე.

2D Transforms:

.transform-examples { transform: translate(50px, 100px); /* ელემენტის გადატანა */ transform: rotate(45deg); /* ელემენტის ბრუნვა */ transform: scale(1.5); /* ელემენტის მასშტაბირება */ transform: skew(20deg, 10deg); /* ელემენტის დახრილება */ } /* რამდენიმე transform ერთად */ .multi-transform { transform: translate(50px, 50px) rotate(45deg) scale(1.2); }

3D Transforms:

.transform-3d { transform: rotateX(45deg); transform: rotateY(45deg); transform: translateZ(100px); transform: perspective(1000px) rotateY(45deg); }

CSS Transitions:

Transitions უზრუნველყოფს მშვიდ ანიმაციებს თვისებების ცვლილებებს შორის.

🎵 HTML5 Audio და Video

📝 პრაქტიკული დავალებები

დავალებების შესრულება დაგეხმარებათ უკეთ გაიგოთ და გამოიყენოთ სესიის განმავლობაში განხილული კონცეფციები.

დავალება 1: CSS Grid განლაგების შექმნა

შექმენით ვებგვერდის განლაგება CSS Grid-ის გამოყენებით, რომელიც შეიცავს ჰედერს, ნავიგაციას, მთავარ კონტენტს და ფუტერს. გამოიყენეთ grid-template-areas ელემენტების დასაწყობად.

დავალება 2: პოზიციონირების პრაქტიკა

შექმენით გვერდი, სადაც სხვადასხვა ელემენტები იყენებენ position: relative, absolute, fixed და sticky. დაამატეთ ინტერფეისის ელემენტები რომლებიც ინტერაქტიულად ცვლიან პოზიციას.

დავალება 3: Transform და Transitions

შექმენით გალერეა სადაც გამოსახულებები გამოიყენებენ transform: scale და rotate ეფექტებს hover-ზე. დაამატეთ რბილი გადასვლები transform თვისებებზე.

დავალება 4: CSS Animation

შექმენით ანიმირებული ლოგო ან ბანერი keyframes-ის გამოყენებით. ანიმაციამ უნდა შეიცავდეს მოძრაობას, ფერის ცვლილებას და ზომის ცვლილებას.

დავალება 5: CSS Variables გამოყენება

შექმენით ვებგვერდი რომელიც იყენებს CSS ცვლადებს თემის ფერების, შრიფტების ზომებისა და სივრცეების განსაზღვრისთვის. დაამატეთ ღილაკი რომელიც შეცვლის თემის ფერებს დინამიურად.
